## 🎨 Дизайн и Стиль

### Карточка концерта:
```tsx
- Размеры: h-40 xs:h-48 sm:h-56 (изображение)
- Скругление: rounded-2xl sm:rounded-3xl
- Border: border-white/10 → hover:border-[#FF577F]/50
- Hover: scale: 1.02, translateY: -5px
- Image: scale: 1.1 при hover
```

### Адаптивность:
- **Mobile (< 640px):** 1 колонка, компактные карточки
- **Tablet (640px+):** 2 колонки
- **Desktop (1024px+):** 3 колонки

### Анимации:
- Staggered появление карточек: `delay: index * 0.1`
- Hover эффекты на карточках
- Scale анимация на кнопке "Купить билет"
- Smooth transitions

## 🔗 Интеграция с Backend

Для подключения к реальным данным:

### 1. Создать API route:
```typescript
// /supabase/functions/server/concerts-routes.tsx
app.get('/concerts/upcoming', async (c) => {
  const concerts = await getConcerts({ status: 'approved', limit: 6 });
  return c.json(concerts);
});
```

### 2. Обновить компонент:
```typescript
// ConcertsSection.tsx
const [concerts, setConcerts] = useState<Concert[]>([]);

useEffect(() => {
  fetch('/api/concerts/upcoming')
    .then(res => res.json())
    .then(data => setConcerts(data));
}, []);
```
