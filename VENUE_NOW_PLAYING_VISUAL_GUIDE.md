## 🔧 КАК ЭТО РАБОТАЕТ (Простыми словами)

### Шаг 1: Venue включает музыку

```
Ты в своем кабинете:
├─ Открываешь плеер
├─ Выбираешь трек "Miles Davis - So What"
├─ Нажимаешь ▶️ Play
└─ Музыка играет в заведении
```

### Шаг 2: Система запоминает

```
Backend (автоматически):
├─ Сохраняет: "Сейчас играет So What"
├─ Сохраняет: Позицию 2:34 / 9:22
├─ Сохраняет: Статус "playing"
└─ Проверяет настройку "showCurrentlyPlaying"
```

### Шаг 3: Посетители видят (если настройка ВКЛ)

```
Публичные страницы:
├─ Каталог заведений → "🎵 Miles Davis - So What"
├─ Мобильное приложение → "🎵 Miles Davis - So What"
├─ Профиль venue → Полный плеер с обложкой
└─ Карта города → "📍 Sunset + 🎵 Miles Davis"
```

## 🎨 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### База данных (1 таблица):

```sql
venue_playback_status:
├─ venue_id: "venue-123"
├─ current_track_name: "So What"
├─ current_artist: "Miles Davis"
├─ current_album: "Kind of Blue"
├─ current_position: 154 (секунд)
├─ duration: 562 (секунд)
├─ is_playing: true
├─ started_at: "2026-02-04T18:30:00Z"
└─ public_visibility: TRUE ← НАСТРОЙКА!
```

### API (2 эндпоинта):

```typescript
// Для venue кабинета (всегда работает)
POST /venue/playback/play
GET  /venue/playback/status

// Для публичного доступа (если настройка ВКЛ)
GET  /public/venues/{id}/now-playing
GET  /public/venues/nearby
```

### Обновление (Real-time):

```
WebSocket → Каждые 10 секунд
├─ Обновляет позицию трека
├─ Обновляет статус (play/pause)
└─ Синхронизирует все публичные страницы
```

## 📊 СРАВНЕНИЕ: ВКЛ vs ВЫКЛ

### ✅ Настройка ВКЛЮЧЕНА:

```
Посетитель видит:
✅ "🎵 Сейчас играет: Miles Davis - So What"
✅ Прогресс: 2:34 / 9:22
✅ Жанр: Jazz
✅ Альбом: Kind of Blue
✅ Анимация звуковых полосок ┃┃┃
✅ Обложка альбома

Результат:
→ Привлекает любителей джаза
→ Показывает атмосферу заведения
→ Выделяется среди конкурентов
→ Увеличивает посещаемость
```

### ❌ Настройка ВЫКЛЮЧЕНА:

```
Посетитель видит:
❌ "🔒 Музыкальная информация скрыта"

Результат:
→ Нет привлечения через музыку
→ Теряешь конкурентное преимущество
→ Меньше посетителей
```

## ✅ РЕЗЮМЕ

### ❓ Вопрос:
"Показывать текущий трек публично - это как?"

### ✅ Ответ:

**ГДЕ:**
1. Каталог заведений (карточки)
2. Мобильное приложение "Nearby"
3. Публичный профиль venue
4. Карта города (будущее)

**КАК:**
1. Ты включаешь трек в плеере
2. Система сохраняет данные
3. Посетители видят "🎵 Сейчас играет: Miles Davis - So What"

**ЗАЧЕМ:**
1. Привлечь любителей твоего жанра
2. Показать атмосферу заведения
3. Выделиться среди конкурентов
4. Увеличить посещаемость

**ПРИМЕР:**
```
Посетитель → Приложение → Nearby
↓
"Sunset Lounge: Miles Davis - So What"
↓
"Люблю джаз!"
↓
Идет в бар! 🎉
```

**Это РЕАЛЬНО работает!**

Тысячи заведений используют такой подход:
- Spotify показывает что слушают друзья
- Instagram показывает где друзья сейчас
- Foursquare показывает популярные места

**Теперь это есть в promo.music для venue!** 🎵✨
